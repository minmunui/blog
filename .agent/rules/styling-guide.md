목적: Shadcn의 깔끔한 색감과 다크/라이트 모드 기능을 구현합니다.

"Tailwind CSS를 사용하여 Shadcn UI 스타일의 디자인 시스템을 구축해 줘.

tailwind.config.js에 Shadcn의 기본 색상 팔레트(Slate/Zinc 계열)를 CSS 변수 방식으로 등록해 줘.

class 전략을 사용하여 다크 모드와 라이트 모드를 전환할 수 있도록 설정해 줘.

사용자의 OS 테마를 기본으로 따르되, 사용자가 수동으로 토글할 수 있는 간단한 JavaScript 스크립트를 작성해 줘.

전체 레이아웃에 적용될 기본 Typography(폰트, 자간 등) 설정을 Shadcn과 유사하게 맞춰줘."
